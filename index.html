저는 안정적이고 확장 가능한 웹 애플리케이션을 만드는 것을 즐기는 Java Spring 개발자입니다.

소개

3년차 Java Spring 개발자 한주승입니다. 저는 주로 Spring Boot, JPA, QueryDSL을 사용하여 백엔드 시스템을 개발하고 있습니다. 대용량 트래픽 처리와 MSA(Microservice Architecture)에 관심이 많으며, 클린 코드와 테스트 코드 작성을 중요하게 생각합니다.

새로운 기술을 배우고 적용하는 것을 좋아하며, 동료들과의 협업을 통해 더 나은 결과물을 만들어내는 과정에서 보람을 느낍니다. 현재는 사이드 프로젝트로 Spring Cloud를 이용한 MSA 기반의 이커머스 플랫폼을 개발하고 있습니다.
